# 网上衣橱系统（Online Wardrobe System）

## 项目简介

网上衣橱系统是一个基于 Vue3 + Spring Boot + MySQL 开发的前后端分离电商管理系统，实现了用户浏览商品、购物车管理、订单管理以及管理员后台管理等功能。

本项目模拟真实服装商城业务流程，包含用户端与管理员端两个子系统，具备完整的数据增删改查功能。

# 数据库设计

项目主要包含以下数据表：

* 用户表（t_user）
* 商品表（t_clothes）
* 商品类型表（t_type）
* 商品尺码表（t_size）
* 购物车表（t_cart）
* 订单表（t_order）
* 订单详情表（t_order_detail）

通过外键关联实现用户、商品、购物车和订单之间的数据关系。

# 项目运行

## 后端启动

```bash
mvn spring-boot:run
```

默认端口：

```text
8080
```

## 前端启动

```bash
npm install
npm run dev
```

默认端口：

```text
7070
```
